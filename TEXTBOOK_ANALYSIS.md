## Детайлни типове задачи от учебника

### ТЕМА 5: КВАДРАТНИ УРАВНЕНИЯ (Стр. 94-117)

#### Непълни квадратни уравнения (Урок 40)

**Тип 1: ax² + bx = 0 (липсва c)**
- Метод: Разлагане на множители - x(ax + b) = 0
- Решения: x₁ = 0; x₂ = -b/a
- Пример: 2x² - 5x = 0 → x(2x - 5) = 0 → x₁ = 0; x₂ = 2.5

**Тип 2: ax² + c = 0 (липсва b)**
- Метод: Решаване чрез корен квадратен
- При c/a < 0: x² = -c/a → x = ±√(-c/a)
- При c/a > 0: x² + c/a > 0 за всяко x (няма решение)
- Пример: x² - 2 = 0 → x = ±√2

**Тип 3: ax² = 0 (липсват b и c)**
- Решение: x = 0 (двоен корен)
- Пример: 3x² = 0 → x₁ = x₂ = 0

#### Пълни квадратни уравнения (Урок 41)

**Формула за корените:**
- x₁,₂ = (-b ± √D) / 2a, където D = b² - 4ac (дискриминанта)

**Три случая:**
1. D > 0: Две различни реални корена
2. D = 0: Един двоен корен x = -b/2a
3. D < 0: Няма реални корени

**Примери:**
- 2x² - 5x - 3 = 0 → D = 49 → x₁ = 3; x₂ = -1/2
- -4x² + 12x - 9 = 0 → D = 0 → x₁ = x₂ = 3/2
- 8x² - 5x + 7 = 0 → D < 0 → няма решение

#### Бикватдратни уравнения (Урок 44)

**Форма:** ax⁴ + bx² + c = 0

**Метод:** Замяна y = x²
1. Получава се квадратно уравнение: ay² + by + c = 0
2. Решаване на квадратното уравнение
3. За всяко положително y: x = ±√y
4. За отрицателно y: няма решение

**Пример:** x⁴ - 10x² + 9 = 0
- Замяна: y² - 10y + 9 = 0
- Решения: y₁ = 9; y₂ = 1
- Обратно: x² = 9 → x = ±3; x² = 1 → x = ±1
- Окончателни решения: x = 3, -3, 1, -1

#### Уравнения от по-висока степен, свеждащи се до квадратни (Урок 45-46)

**Метод:** Разлагане на множители или подходяща замяна

**Пример:** 4(x-3)² - 3(x+4)(x+2) = 12
- Разширяване и опростяване до квадратно уравнение
- Решаване чрез формула

### ТЕМА 7: РАЦИОНАЛНИ ИЗРАЗИ (Стр. 148-169)

#### Дробни уравнения (Урок 72-73)

**Метод:**
1. Определяне на дефиниционното множество (знаменателите ≠ 0)
2. Привеждане към общ знаменател
3. Решаване на полученото уравнение
4. Проверка дали решенията са в дефиниционното множество

**Пример:** (x+10)/2 - x(x-2)/3 = 5
- Дефиниционно множество: всички реални числа
- Привеждане към общ знаменател 6
- Решаване на полученото квадратно уравнение

### ТЕМА 4: КВАДРАТЕН КОРЕН (Стр. 72-91)

#### Операции с квадратни корени

**Основни свойства:**
- √(a·b) = √a · √b (за a, b ≥ 0)
- √(a/b) = √a / √b (за a ≥ 0, b > 0)
- √(a²) = |a|
- (√a)² = a (за a ≥ 0)

**Рационализация на знаменател:**
- 1/√a = √a/a
- 1/(√a + √b) = (√a - √b)/(a - b)
