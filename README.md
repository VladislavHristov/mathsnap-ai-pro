# MathSnap AI Pro

**Решавай математически задачи с помощта на AI**

MathSnap AI Pro е мобилно приложение, което използва изкуствен интелект за разпознаване и решаване на математически задачи от снимки. Просто направи снимка на задачата или качи изображение, и приложението ще ти покаже подробното решение със всички стъпки.

---

## 🎯 Основни функции

### 📸 Снимане на задачи
- **Камера**: Направи снимка на математическа задача директно от приложението
- **Галерия**: Качи съществуващо изображение от твоя телефон
- **Автоматично компресиране**: Изображенията се оптимизират за бърза обработка

### 🤖 AI анализ
- **Google Vision OCR**: Разпознава текст и математически символи от снимките
- **OpenAI класификация**: Определя типа на задачата (алгебра, геометрия, анализ и т.н.)
- **Оценка на сложност**: Определя нивото на трудност (лесно, средно, трудно)
- **Избор на решавач**: Автоматично избира подходящия решавач (OpenAI или Wolfram Alpha)

### 📝 Подробни решения
- **Стъпка по стъпка**: Всяко решение е разбито на ясни логични стъпки
- **Обяснения**: Всяка стъпка има подробно обяснение
- **Формули**: Всички използвани формули са видими
- **Окончателен отговор**: Ясно маркиран крайния резултат

### 💾 История и любими
- **История на решени задачи**: Всички решени задачи се запазват автоматично
- **Любими**: Отбележи важни задачи за бърз достъп
- **Търсене и филтриране**: Намери лесно старите решения

### 🎨 Персонализация
- **Тъмен режим**: Удобен за очите в нощни часове
- **Избор на език**: Поддържа български и английски
- **Управление на кеш**: Изчисти старите данни за освобождаване на място

## 📱 Екрани на приложението

### Начален екран (Home)
Показва преглед на решени задачи днес, любимите задачи и бързи действия за снимане или качване на изображение.

**Функции:**
- Брой решени задачи днес
- Брой любими задачи
- Бутон "Направи снимка"
- Бутон "Качи изображение"
- Последни решени задачи

### Екран за камера (Camera)
Позволява снимане на задачи или избор от галерия.

**Функции:**
- Снимане с камерата
- Избор от галерия
- Статус на компресиране
- Индикатор на прогреса

### Екран за обработка (Processing)
Показва статуса на анализа на задачата.

**Функции:**
- Индикатор на прогреса
- Статус съобщения
- Обработка на OCR
- Класификация на задачата
- Решаване

### Екран със решение (Solution)
Показва подробното решение на задачата.

**Функции:**
- Извлечена задача
- Тип на задачата
- Ниво на сложност
- Решавач (OpenAI / Wolfram Alpha)
- Стъпка по стъпка решение
- Окончателен отговор
- Бутон за добавяне в любими
- Бутон за споделяне

### История (History)
Показва всички решени задачи.

**Функции:**
- Списък на всички решени задачи
- Дата на решаване
- Тип на задачата
- Търсене и филтриране
- Изтриване на задачи
- Pull-to-refresh

### Любими (Favorites)
Показва отбелязаните задачи.

**Функции:**
- Списък на любимите задачи
- Бърз достъп до решения
- Премахване от любими

### Настройки (Settings)
Управление на приложението.

**Функции:**
- Тъмен/светъл режим
- Избор на език (български/английски)
- Изчистване на кеш
- Информация за приложението
- Версия на приложението

## 🚀 Как да използваш приложението

### Решаване на задача

1. **Отвори приложението** и натисни "Направи снимка" или "Качи изображение"
2. **Направи снимка** на математическа задача или избери изображение от галерия
3. **Чакай анализа** - приложението ще обработи снимката
4. **Преглед на решението** - виж подробното решение със всички стъпки
5. **Добави в любими** (по желание) - отбележи важни задачи

### Преглед на история

1. Отвори раздел "История"
2. Преглед всички решени задачи
3. Натисни на задача за преглед на решението
4. Използвай търсене за намиране на конкретна задача

### Управление на настройки

1. Отвори раздел "Настройки"
2. Включи/изключи тъмния режим
3. Избери език (български или английски)
4. Изчисти кеш ако приложението работи бавно

## ⚙️ Инсталация и разработка

### Изисквания
- Node.js 18+ и pnpm
- Expo CLI
- Android SDK или iOS Xcode (за локално тестване)

### Инсталация

```bash
# Клониране на хранилището
git clone <repository-url>
cd mathsnap-ai-pro

# Инсталиране на зависимостите
pnpm install

# Стартиране на dev сървър
pnpm dev

# За Android
pnpm android

# За iOS
pnpm ios
```

### Конфигурация на API ключове

Приложението изисква следните API ключове:
- **Google Vision API Key** - за OCR разпознаване
- **OpenAI API Key** - за класификация и решаване
- **Wolfram Alpha AppID** - за символни изчисления

Добави ги в файла `.env`:

```
GOOGLE_VISION_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
WOLFRAM_APP_ID=your_app_id_here
```

## 📊 Архитектура

### Backend
- **Express.js сървър** с tRPC API
- **Google Vision API** за OCR
- **OpenAI GPT-4** за класификация и решаване
- **Wolfram Alpha API** за символни изчисления
- **MySQL база данни** за хранене на история

### Frontend
- **React Native** с Expo
- **Expo Router** за навигация
- **AsyncStorage** за локално хранилище
- **TanStack Query** за управление на API заявки
- **Tailwind CSS** за стилизиране

## 🐛 Известни проблеми и решения

| Проблем | Решение |
|---------|---------|
| Приложението е бавно | Изчисти кеш в настройките |
| Камерата не работи | Проверка на разрешенията за камера в настройките на телефона |
| Снимката не се разпознава | Направи по-ясна снимка с добро осветление |
| Решението е неточно | Проверка дали задачата е ясно видима на снимката |
